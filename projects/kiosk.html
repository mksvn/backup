Michelle Meksavanh (UX/UI Designer)

Order system tailored for casino buffets

Client
Aurora Data Logic is a software engineering company that specializes in custom point of sales systems.

Objective
Design a kiosk ordering system centered around a casino buffet host's guest check-in process and optimized for tablets.

User Stories
  • As a buffet host, I want to add guests to the bill as either adults, seniors or children, so they are charged the appropriate amounts.
  • As a buffet host, I want to review an order before payment, so that I can make adjustments if needed.
  • As a buffet host, I want to scan a player's card and accept card payments, so I can apply discounts and close the check.
