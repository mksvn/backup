Michelle Meksavanh (UX/UI Designer)

On the go hands-free communication

Client
HeyDay App is a push notification service used by engineers to send their app communications to wearable devices.

My Role
Provide various deliverables using the 'minimum viable product strateg' to abide by the restraints of time and budget.

Creative Direction, Visual Design, Front-End Development
  • Create branding that includes of a logo, illustrations, style guide (mvp) and UI design system (mvp)
  • Create social media assets, a landing page and email template with new branding
UX/UI Design
  • Design a web dashboard where engineers can create message channels and subscribe users to them.
  • Design a mobile app where users can receive messages from channels. Mock up the app on wearOS and watchOS.
  • As an engineer, I want to create channels and subscribed users to it, so that I can feed messages into the channel and notify users.
  • As an engineer, I want to see an overview of channels, so that I can review them all at once.
  • As an engineer, I want to deactivate channels, so that I can pause or stop sending messages.
  • As a subscribed user, I want to be notified of new messages on my iPhone, Apple watch or Android watch, so that I can address them immediately.

Combines the visuals of:

  1. A message bubble (elipses used to emphasize that the top portion is a talk bubble)
  2. The letter H
  3. Watch on a wrist, the bottom point resembles cuffs on dress shirt
